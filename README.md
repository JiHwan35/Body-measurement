## 프로젝트 목표

1. 사용자가 직접 촬영한 사진을 기반으로 Body-Pix의 Body segmentation을 통해서 얻은 픽셀 정보를 
   이용하여 신체 치수 정보를 사용자에게 제공 

2. 사용자가 직접 촬영한 사진을 기반으로 딥러닝 알고리즘으로 측정된 신체 치수 정보를 사용자에게 제공 

3. 어깨 너비, 가슴 둘레, 총장 (상체 길이), 허리너비, 총장 (하체 길이)등 여러 치수를 사용자에게 출력
